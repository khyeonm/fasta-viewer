# fasta-viewer

Sequence viewer with colored nucleotides, sequence statistics, and GC content calculation.

## Features

- Colored nucleotide display (A=green, T=red, G=gold, C=blue)
- Auto-detects DNA, RNA, or protein sequences
- Per-sequence GC content calculation
- Sequence length and count summary
- Text search across sequence headers
- Sequence detail view with formatted base display
- Pagination for files with many sequences (50 sequences per page)

## Supported Extensions

- `.fasta`
- `.fa`

## Installation

Install from the **Plugins** tab in the AutoPipe desktop app.
